Convenções para este treinamento

Linguagens utilizadas

Durante este treinamento, utilizaremos duas linguagens principais:

PHP

A linguagem PHP em sua versão 8 será utilizada pra desenvolvermos o back-end de nossas aplicações.

SQL

Utlizaremos a linguagem SQL (Structured Query Language) para criar e alimentar nossos bancos de dados MySQL.

Adicionalmente, utilizaremos HTML5 e CSS3, para estabelecer as estruturas e melhorar o visual de nossas aplicações.

Nomes de Diretórios e Arquivos

Para a criação de diretórios e arquivos, não utilizaremos letras maiúsculas, espaços ou caracteres especiais no nomes. Ex.:

Flávio → flavio

Flávio Ricardo → flavio-ricardo ou flavio_ricardo

Recursos & Ferramentas

Para o desenvolvimento Back-end em PHP com Banco de Dados MySQL, utilizaremos os seguintes recursos:

  • Computador: Com sistema Operacional Windows 10, com acesso à internet.
  • Servidor Web: Apache HTTP com PHP8 e MySQL8
  • Navegador (Browser): Google Chrome, Micosoft Edge ou Mozilla Firefox
  • Editor de Código: Microsoft Visual Studio Code, Sublime Text ou equivalente.
  • Sistema de Versionamento e Gerenciamento de Repositórios: Git e GitHub.
