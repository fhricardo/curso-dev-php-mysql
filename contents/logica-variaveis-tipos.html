Aula 3 – Variáveis e Tipos de Dados

Variáveis

Ao escrever algoritmos, vimos que precisamos guardar informações temporárias para utilizá-las durante o processamento. No PHP (assim como em outras linguagens), esses valores são armazenados em variáveis.

Uma variável é como uma caixa com nome, onde guardamos dados que podem ser consultados e modificados. No PHP, toda variável é identificada pelo símbolo $ antes do nome. Exemplos: $idade, $nome, $aprovado.

Tipos

No PHP, as variáveis não precisam ser declaradas com tipo fixo (a linguagem é fracamente tipada), mas o valor que você atribui define o tipo de dado que ela armazenará.

Os principais tipos de dados em PHP são:

  • int (inteiro) – números sem casas decimais.
    Exemplo: $idade = 25;
  • float (ou double, real) – números com casas decimais.
    Exemplo: $altura = 1.75;
  • string (texto) – conjunto de caracteres (palavras, frases, símbolos).
    Exemplo: $nome = "Carlos";
  • bool (booleano) – valores lógicos verdadeiro ou falso.
    Exemplo: $aprovado = true;
  • array – conjunto de valores organizados em lista.
    Exemplo: $cores = ["azul", "verde", "vermelho"];
  • null – representa ausência de valor.
    Exemplo: $telefone = null;

Exemplo em PHP juntando alguns desses tipos:

<?php
    $nome = "Ana";       // string
    $idade = 22;         // int
    $altura = 1.68;      // float
    $aprovado = true;    // bool
    $materias = ["PHP", "MySQL", "HTML"]; // array

    echo "Nome: $nome";  
    echo "Idade: $idade anos";
    echo "Altura: $altura m";
    echo "Aprovado? " . ($aprovado ? "Sim" : "Não");
?>

Compreender os tipos de dados é essencial para evitar erros e manipular informações corretamente. Embora o PHP faça a conversão de tipos automaticamente em muitos casos, é importante ter clareza sobre o que cada variável está armazenando. Nas próximas aulas, vamos utilizar essas variáveis em estruturas de decisão e repetição, tornando nossos programas cada vez mais dinâmicos.
