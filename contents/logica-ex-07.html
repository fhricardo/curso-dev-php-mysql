Aula 07 - Exercício: Operadores Relacionais

Objetivo: Praticar o uso dos operadores relacionais em PHP.

Enunciado:

Crie um arquivo chamado relacionais.php que faça o seguinte:

  1. Defina duas variáveis, por exemplo $a e $b, com valores diferentes (um inteiro e uma string numérica, como 10 e "10").
  2. Utilize os operadores relacionais abaixo e exiba o resultado de cada comparação com var_dump():
    • Igualdade (==)
    • Idêntico (===)
    • Diferente (!= ou <>)
    • Não-Idêntico (!==)
    • Maior que (>)
    • Menor que (<)
    • Maior ou Igual (>=)
    • Menor ou Igual (<=)
  3. Mostre os resultados no formato: $a == $b : bool(true)
  4. Por fim, teste também o operador spaceship (<=>) em três situações diferentes:
    • Comparando dois números inteiros
    • Comparando dois números decimais
    • Comparando duas strings

Exemplo de saída esperada:

$a == $b : bool(true)
$a === $b : bool(false)
$a != $b : bool(false)
$a !== $b : bool(true)
$a > $b : bool(false)
$a < $b : bool(false)
$a >= $b : bool(true)
$a <= $b : bool(true)

1 <=> 2 : -1
2.5 <=> 1.5 : 1
"a" <=> "b" : -1

Use var_dump() para ver claramente os valores booleanos (true ou false) retornados pelas comparações.
