Aula 1 – Comandos de Saída em PHP

Os comandos de saída são essenciais para a exibição de resultados ou de informações detalhadas para que possamos verificar o correto funcionamento dos nossos códigos.

No PHP, os comandos de saída mais utilizados são:

  • echo – exibe um ou mais valores.
  • print – semelhante ao echo, mas retorna um valor (1) e pode ser usado em expressões.
  • print_r() – usado para exibir arrays e informações de forma legível.
  • var_dump() – mostra informações detalhadas sobre variáveis (tipo e valor).

Comando echo

O comando echo é o mais simples e direto para mostrar dados na tela.

<?php
    echo "Olá, mundo!";
    echo "Bem-vindo ao curso de PHP!";
?>

Também é possível exibir variáveis com echo:

<?php
    $nome = "Ana";
    $idade = 22;
    echo "Meu nome é $nome e tenho $idade anos.";
?>

Comando print

O comando print funciona de forma semelhante ao echo, mas tem uma pequena diferença: ele sempre retorna o valor 1. Por isso, pode ser usado em expressões.

<?php
    print "Este texto foi exibido com print.";
?>

Comando print_r

O print_r() é muito útil para visualizar o conteúdo de arrays de maneira mais legível.

<?php
    $materias = ["PHP", "MySQL", "HTML"];
    print_r($materias);
?>

Comando var_dump

O comando var_dump() exibe detalhes completos sobre uma variável, incluindo o tipo de dado e o valor. É muito usado para depuração (debug).

<?php
    $idade = 22;
    var_dump($idade);
    $materias = ["PHP", "MySQL", "HTML"];
    var_dump($materias);
?>

Esses comandos de saída serão nossos primeiros aliados no PHP, pois permitem verificar se o código está funcionando corretamente. Com eles, já conseguimos interagir com o navegador e testar variáveis, textos e dados em geral.
