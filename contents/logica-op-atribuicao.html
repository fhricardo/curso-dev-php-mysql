Aula 06 - Operadores de Atribuição

Operador de Atribuição

Permite atribuir valores a variáveis. Símbolo utilizado: =

<?php
    $nome = 'Flávio';
    // Neste caso, lemos a linha acima da seguinte forma:
    // A variável nome recebe o valor 'Flávio'

Operadores de Atribuição Aritmética

também conhecidos como Operadores de Atribuição Reduzida, permitem que, além da atribuição de valores às variáveis, também sejam realizadas operações aritméticas

Atribuição com Soma

Símbolo utilizado: +=

<?php
    $a = 10;
    $b = 5;
    $a += $b; //Equivale a $a = $a + $b;
    echo $a;  //Exibe 15

Atribuição com Subtração

Símbolo utilizado: -=

<?php
    $a = 10;
    $b = 5;
    $a -= $b; //Equivale a $a = $a - $b;
    echo $a;  //Exibe 5

Atribuição com Multiplicação

Símbolo utilizado: *=

<?php
    $a = 10;
    $b = 5;
    $a *= $b; //Equivale a $a = $a * $b;
    echo $a;  //Exibe 50

Atribuição com Divisão

Símbolo utilizado: /=

<?php
    $a = 10;
    $b = 5;
    $a /= $b; //Equivale a $a = $a / $b;
    echo $a;  //Exibe 2

Atribuição com Módulo

Símbolo utilizado: %=

<?php
    $a = 10;
    $b = 5;
    $a %= $b; //Equivale a $a = $a % $b;
    echo $a;  //Exibe 0

Atribuição com Exponenciação

Símbolo utilizado: **=

<?php
    $a = 10;
    $b = 5;
    $a **= $b; //Equivale a $a = $a ** $b;
    echo $a;  //Exibe 100000

Operadores de Concatenação

Concatenação

Permite que os valores de duas ou mais variáveis ou strings sejam apresentados como um único valor. Símbolo utilizado .

Este operador é muito utilizado para compor frases de saída (strings) utilizando trechos de textos e valores de strings.

<?php
    $nome = 'Flávio';
    $idade = 44;
    $saida = $nome . " tem " . $idade . " anos.";
    echo $saida;  //Exibe: Flávio tem 44 anos.

Atribuição com Concatenação

Permite atribuir à uma variável o valor dela concatenado com outro valor. Símbolo utilizado .=

<?php
    $a = 10;
    $b = 10;
    $a .= $b; //Equivale a $a = $a . $b;
    echo $a;  //Exibe 1010

Opreador de Coalescência Nula

O operador de coalescência nula foi adicionado na versão 7 do PHP como um truque sintático para o caso trivial de precisar usar um ternário em conjunto com a função isset(). Ele retorna o primeiro operando se este existir e não for null; caso contrário retorna o segundo operando. Símbolo utilizado: ??

<?php
    $a = 5;
    $b = 10;
    $saida = $a ?? $b;
    echo $saida; // Mostra 5, pois a variável $a foi declarada e não é nula.

    $b = 10;
    $saida = $a ?? $b;
    echo $saida; // Mostra 10, pois a variável $a não foi declarada.

    $a;
    $b = 10;
    $saida = $a ?? $b;
    echo $saida; // Mostra 10, pois a variável $a foi declarada mas, sem um valor atribuído, ou seja, é nula.
