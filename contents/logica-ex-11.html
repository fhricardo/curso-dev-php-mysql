Aula 11 - Exercício: Formulário com Estruturas Condicionais

Objetivo: Desenvolver um sistema que recebe informações de um formulário HTML, processa com PHP e utiliza condicionais para exibir mensagens personalizadas.

Enunciado:

Você foi contratado pela Power Fit para criar um sistema de cadastro de novos alunos. O sistema deve receber os dados de um formulário HTML e processá-los em PHP, mostrando os resultados com base em cálculos e estruturas condicionais.

Regras do Sistema:

  1. O formulário deve conter os seguintes campos:
    • Idade (campo numérico)
    • Peso (campo numérico, em kg)
    • Altura (campo numérico, em metros, ex: 1.75)
    • Plano (campo de seleção: 1 - Mensal, 2 - Trimestral, 3 - Anual)
    • Botão de envio
  2. O sistema deve:
    • Exibir a categoria do aluno pela idade:
      • < 18: "Categoria Jovem"
      • 18 a 40: "Categoria Adulto"
      • > 40: "Categoria Master"
    • Calcular o IMC com a fórmula: IMC = peso / (altura * altura)
    • Exibir a classificação do IMC:
      • < 18,5: "Abaixo do peso"
      • 18,5 a 24,9: "Peso normal"
      • 25 a 29,9: "Sobrepeso"
      • >= 30: "Obesidade"
    • Exibir a descrição do plano escolhido usando switch:
      • 1: "Plano Mensal: flexibilidade total."
      • 2: "Plano Trimestral: foco em resultados de curto prazo."
      • 3: "Plano Anual: máximo custo-benefício."
      • Qualquer outro: "Plano inválido."
  3. No final, mostre o valor do plano:
    • Mensal = R$ 120
    • Trimestral = R$ 300
    • Anual = R$ 1000
    Se o aluno tiver IMC acima de 25, aplicar 20% de desconto.

Desafio Extra

Altere o sistema para que o formulário e os resultados fiquem na mesma página. Exiba também uma mensagem de erro caso o peso ou a altura sejam iguais a zero (evitando divisão por zero no cálculo do IMC).
