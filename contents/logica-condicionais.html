Aula 10 – Estruturas Condicionais em PHP

As estruturas condicionais permitem que nossos programas tomem decisões de acordo com determinadas condições. No PHP, temos três formas principais de estruturar decisões: if, if...else e switch.

Estrutura if

O if é usado quando queremos executar um bloco de código apenas se uma condição for verdadeira. 

<?php
$idade = 20;

if ($idade >= 18) {
    echo "Você é maior de idade.";
}

Estrutura if...else

O if...else permite escolher entre dois caminhos: se a condição for verdadeira, executa o primeiro bloco; caso contrário, executa o bloco alternativo. 

<?php
$idade = 16;

if ($idade >= 18) {
    echo "Você pode dirigir.";
} else {
    echo "Você ainda não pode dirigir.";
}

Estrutura if...elseif...else

Quando temos mais de duas possibilidades, podemos usar elseif para testar múltiplas condições em sequência. 

<?php
$nota = 7;

if ($nota >= 9) {
    echo "Excelente!";
} elseif ($nota >= 7) {
    echo "Bom!";
} elseif ($nota >= 5) {
    echo "Recuperação!";
} else {
    echo "Reprovado!";
}

Estrutura switch

O switch é usado quando precisamos comparar uma mesma variável com vários valores possíveis. É uma alternativa ao uso de vários elseif. 

<?php
$dia = "terca";

switch ($dia) {
    case "segunda":
        echo "Início da semana!";
        break;
    case "terca":
        echo "Segundo dia útil!";
        break;
    case "quarta":
        echo "Metade da semana!";
        break;
    case "sexta":
        echo "Sextou!";
        break;
    default:
        echo "Dia comum.";
}

O uso de break no switch é essencial para evitar que todos os casos seguintes sejam executados. Caso queira que isso aconteça de propósito, basta omitir o break.

As estruturas condicionais são fundamentais para qualquer programa, pois permitem criar regras, validar informações e controlar o fluxo de execução. Com elas, nossos scripts deixam de ser lineares e passam a responder a diferentes situações.
