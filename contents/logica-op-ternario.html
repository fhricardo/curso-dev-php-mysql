Aula 09 - Operador Ternário

O operador ternário é uma forma simplificada de escrever estruturas if...else. Ele recebe esse nome porque utiliza três partes em sua expressão: uma condição, um resultado para true e um resultado para false.

A sintaxe é:

condição ? valor_se_verdadeiro : valor_se_falso;

Exemplo básico

Suponha que você queira verificar a maioridade de uma pessoa. Com if...else ficaria assim: 

<?php
$idade = 20;

if ($idade >= 18) {
    echo "Maior de idade";
} else {
    echo "Menor de idade";
}

Usando o operador ternário, podemos escrever a mesma lógica em uma única linha:

<?php
$idade = 20;
echo ($idade >= 18) ? "Maior de idade" : "Menor de idade";

Atribuindo valores

O operador ternário pode ser usado também para definir o valor de uma variável de acordo com uma condição: 

<?php
$nota = 7;
$status = ($nota >= 6) ? "Aprovado" : "Reprovado";

echo "Resultado: $status";

Operador Ternário Aninhado

É possível aninhar operadores ternários, mas é preciso cuidado para não prejudicar a legibilidade. Por exemplo: 

<?php
$nota = 8;

$resultado = ($nota >= 7) ? "Aprovado" : 
            (($nota >= 5) ? "Recuperação" : "Reprovado");

echo $resultado;

Nesse caso: se a nota for maior ou igual a 7 → aprovado. Senão, se for maior ou igual a 5 → recuperação. Caso contrário → reprovado.

O operador ternário é ótimo para simplificar código, mas evite exageros. Se a expressão ficar muito longa ou difícil de entender, prefira usar if...else tradicional.

Com o operador ternário, ganhamos uma forma curta e prática de escrever condições, tornando nosso código mais enxuto e expressivo.
