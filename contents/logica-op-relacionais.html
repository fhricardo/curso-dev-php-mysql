Aula 07 - Operadores de Relacionais

O que são Operadores Relacionais?

São os operadores utilizados para comparar (ou relacionar) valores fixos ou de variáveis e retornar um resultado, na maioria das vezes booleano (true ou false).

Operador Igualdade

Utilizado para comparar se dois valores são iguais. Símbolo utilizado: ==

Este operador compara os valores mas ignora o tipo.

<?php
    $a = 10;
    $b = "10";
    var_dump($a == $b); //Retorna bool(true)

Operador Idêntico

Utilizado para comparar se dois valores são idênticos. Símbolo utilizado: ===

Este operador compara os valores e o tipo, retornando true apenas se os dois parâmetros comparados forem verdadeiros.

<?php
    $a = 10;
    $b = "10";
    var_dump($a === $b); //Retorna bool(false), uma vez que $a é do tipo int e $b é do tipo string

Operador Igualdade

Utilizado para comparar se dois valores são iguais. Símbolo utilizado: ==

Este operador compara os valores mas ignora o tipo.

<?php
    $a = 10;
    $b = "10";
    var_dump($a == $b); //Retorna bool(false)

Operador Não-Igual (Diferente)

Utilizado para comparar se dois valores não são iguais. Símbolos utilizados: != ou <>

Este operador compara os valores mas ignora o tipo.

<?php
    $a = 10;
    $b = "10";
    var_dump($a != $b); //Retorna bool(false), uma vez que os dois valores são iguais.

    // O mesmo ocorre em

    var_dump($a <> $b); //Retorna bool(false), uma vez que os dois valores são iguais.

Operador Não-dêntico

Utilizado para comparar se dois valores não são idênticos. Símbolo utilizado: !==

Este operador compara os valores e o tipo, retornando true apenas se os dois parâmetros comparados forem falsos.

<?php
    $a = 10;
    $b = "10";
    var_dump($a !== $b); //Retorna bool(true), uma vez que $a é do tipo int e $b é do tipo string

Operador Maior que

Utilizado para comparar se o valor de $a é maior do que o valor de $b. Símbolo utilizado: >

<?php
    $a = 10;
    $b = 5;
    var_dump($a > $b); //Retorna bool(true), uma vez que $a é maior do que $b

Operador Menor que

Utilizado para comparar se o valor de $a é menor do que o valor de $b. Símbolo utilizado: >

<?php
    $a = 10;
    $b = 5;
    var_dump($a < $b); //Retorna bool(false), uma vez que $a é maior do que $b

Operador Maior ou Igual

Utilizado para comparar se o valor de $a é maior ou igual ao valor de $b. Símbolo utilizado: >=

<?php
    $a = 5;
    $b = 5;
    var_dump($a >= $b); //Retorna bool(true), uma vez que $a é igual a $b

Operador Menor ou Igual

Utilizado para comparar se o valor de $a é menor ou igual ao valor de $b. Símbolo utilizado: <=

<?php
    $a = 5;
    $b = 5;
    var_dump($a <= $b); //Retorna bool(true), uma vez que $a é igual a $b

Operador Spaceship (Nave Espacial)

O PHP conta com um operador relacional chamado Spaceship (Nave Espacial). Ao contrário dos outros operadores relacionais, que retornam resultados true ou false, o Spacheship retona os valores -1 , 0 ou 1. Símbolo utilizado: <=>

O operador spaceship é utilizado para comparar duas expressões. Ele retorna -1, 0 ou 1 quando $a for respectivamente menor que, igual a ou maior que $b.

<?php
    // Inteiros
    echo 1 <=> 1; // 0
    echo 1 <=> 2; // -1
    echo 2 <=> 1; // 1

    // Números de ponto flutuante
    echo 1.5 <=> 1.5; // 0
    echo 1.5 <=> 2.5; // -1
    echo 2.5 <=> 1.5; // 1

    // Strings
    echo "a" <=> "a"; // 0
    echo "a" <=> "b"; // -1
    echo "b" <=> "a"; // 1
