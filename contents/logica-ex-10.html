Aula 10 - Exercício: Estruturas Condicionais

Objetivo: Colocar em prática o uso das estruturas condicionais (if, if...else, elseif e switch) para resolver problemas reais.

Enunciado:

Você foi contratado para desenvolver um pequeno sistema de cadastro de usuários para uma plataforma fictícia chamada Calabreso Play. O sistema deve receber algumas informações e, com base nelas, tomar decisões usando estruturas condicionais.

Regras do Sistema:

  1. O usuário informa sua idade.
    • Se tiver < 13: exiba "Você só pode acessar a área Kids."
    • Se tiver entre 13 e 17: exiba "Acesso liberado, mas com conteúdo restrito."
    • Se tiver >= 18: exiba "Acesso total permitido."
  2. O usuário informa sua nota em um teste de cadastro (de 0 a 10).
    • Se nota >= 9: exiba "Usuário nível Platina."
    • Se nota entre 7 e 8: exiba "Usuário nível Ouro."
    • Se nota entre 5 e 6: exiba "Usuário nível Prata."
    • Se nota < 5: exiba "Usuário nível Bronze."
  3. O usuário escolhe o plano de assinatura:
    • 1 – Básico
    • 2 – Premium
    • 3 – Família
    Use switch para exibir:
    • "Plano Básico: acesso limitado a filmes."
    • "Plano Premium: acesso a filmes e séries."
    • "Plano Família: acesso completo + múltiplos perfis."
    • Qualquer outro número: "Plano inválido."

Exemplo de Execução:

Exemplo de Saída no Navegador:

Idade: 15

Acesso liberado, mas com conteúdo restrito.

Nota: 8

Usuário nível Ouro.

Plano escolhido: 2

Plano Premium: acesso a filmes e séries.

Valor a pagar: R$20

Desafio Extra

Peça para o sistema calcular o valor do plano escolhido, aplicando as seguintes regras:

  • Básico = R$ 20
  • Premium = R$ 40
  • Família = R$ 60

Se o usuário tiver menos de 18 anos, aplique um desconto de 50%. Mostre no final: "Valor a pagar: R$XX".
