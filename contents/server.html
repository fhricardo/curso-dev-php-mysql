O Servidor de Desenvolvimento

Para utilizarmos nosso computador como um servidor de desenvolvimento, é necessário realizar a instalação do servidor Web Apache e realizar a sua configuração para que ele possa trabalhar com PHP. Também é necessário instalar o Sistema de Gerenciamento de Banco de Dados MySQL.

O servidor Web Apache

O sistema do servidor Web Apache para Windows pode ser obtido por meio do link: https://www.apachelounge.com/download/. Selecione a versão Apache 2.4.65-250724 Win64 (versão mais recente na última atualização deste conteúdo) e faça o download do arquivo .zip (Clique aqui para fazer o download direto)

Após o download, descompacte o arquivo e renomeie a pasta para Apache24 (sim, com a primeira letra maiúscula!)

Mova a pasta para a raiz da unidade C:\

Visual C++ Redistribuível

Outro recurso necessário ao correto funcionamento do PHP em servidores Web Apache rodando em sistemas operacionais Microsoft Windows é o Visual C++ Redistribuível, que pode ser baixado no site https://learn.microsoft.com/pt-br/cpp/windows/latest-supported-vc-redist?view=msvc-170

Caso prefira o link direto para download, utilize um dos listados abaixo:

  • Para sistemas 64 bits: https://aka.ms/vs/17/release/vc_redist.x64.exe
  • Para sistemas de 32 bits: https://aka.ms/vs/17/release/vc_redist.x86.exe
