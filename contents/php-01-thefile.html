Aula 01 – O Arquivo PHP

Antes de começarmos a escrever programas mais complexos, é importante entender como funciona a estrutura de um arquivo PHP. O PHP é uma linguagem de programação interpretada, ou seja, o código precisa ser processado por um servidor que possua o interpretador PHP instalado.

Um arquivo PHP geralmente tem a extensão .php e pode conter tanto código PHP quanto HTML, permitindo misturar programação com conteúdo da página.

1. Estrutura Básica

Todo código PHP deve ser escrito dentro das tags especiais <?php e ?>

<?php
    // Aqui vai o código PHP
?>

Também é possível escrever código PHP dentro de arquivos HTML.

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Meu primeiro arquivo PHP</title>
</head>
<body>

    <?php
        echo "Olá, mundo!";
    ?>

</body>
</html>

2. Comentários em PHP

Comentários são trechos de texto que não são executados pelo interpretador, mas servem para documentar o código. Existem três formas de escrever comentários em PHP:

<?php
    // Este é um comentário de uma linha
    # Este também é um comentário de uma linha - estilo UNIX
    /* 
    Este é um comentário
    que pode ocupar várias linhas
    */
    echo "Comentários não aparecem na execução!";
?>

3. Boas Práticas

Algumas boas práticas ao criar arquivos PHP:

  • Utilizar sempre a tag completa <?php ?>, evitando a tag curta <? ?>.
  • Organizar o código com indentação e espaçamento adequado.
  • Usar nomes de arquivos claros e relacionados ao seu conteúdo, como index.php ou cadastro.php.
  • Comentar trechos importantes do código para facilitar a manutenção.
  • Separar, sempre que possível, lógica de programação do HTML (seguindo o padrão MVC em projetos maiores).

Compreender a estrutura básica de um arquivo PHP e adotar boas práticas desde o início é fundamental para escrever códigos mais organizados, fáceis de entender e de manter. Agora que já sabemos estruturar nossos arquivos, podemos avançar para a criação de programas mais completos.
