Aula 08 – Operadores Lógicos em PHP

Os operadores lógicos permitem combinar condições em expressões, retornando resultados do tipo boolean (true ou false). Eles são muito utilizados em estruturas de decisão como if, else e while.

abaixo, vamos conhecer os operadores lógicos utilizados no PHP:

Operador NOT

O operador NOT (ou NÃO) inverte o valor lógico de uma expressão: transforma true em false e vice-versa.

O símbolo utilizado é: !

<?php
    $chovendo = false;
    if (!$chovendo) {
        echo "Pode sair sem guarda-chuva.";
    } else {
        echo "Leve o guarda-chuva!";
    }

Operador AND

O operador AND (ou E) retorna true somente se todas as condições forem verdadeiras.

Os símbolos utilizados são: && e and

<?php
    $idade = 20;
    $temCNH = true;
    if ($idade >= 18 && $temCNH) {
    echo "Pode dirigir.";
    } else {
    echo "Não pode dirigir.";
    }

Os dois símbolos, && e and podem ser utilizados em nossos script. A diferença entre eles está na ordem de precedência.

Operador OR

O operador OR (ou OU) retorna true se pelo menos uma das condições for verdadeira.

Os símbolos utilizados são: || e or

<?php
    $feriado = true;
    $fimDeSemana = false;

    if ($feriado || $fimDeSemana) {
        echo "Hoje não tem aula!";
    } else {
        echo "Hoje é dia de estudar.";
    }

Os dois símbolos, || e or podem ser utilizados em nossos script. A diferença entre eles está na ordem de precedência.

Operador XOR (xor)

O XOR (ou “OU exclusivo”) retorna true quando exatamente uma das condições é verdadeira. Se as duas forem true ou as duas forem false, o resultado é false. Em PHP, o operador lógico recomendado é a palavra-chave xor (não confundir com ^, que é bitwise).

Exemplo prático: dar desconto se o cliente tiver cupom ou for assinante, mas não os dois ao mesmo tempo. 

<?php
    $cupom = true;
    $assinante = false;

    if ($cupom xor $assinante) {
    echo "Desconto aplicado!";
    } else {
    echo "Sem desconto (ou nenhum, ou ambos os critérios foram atendidos).";
    }

Atenção à precedência: xor tem precedência baixa em PHP.Ao misturar com atribuição ou outras operações, use parênteses: $resultado = (true xor false);

Com esses operadores, podemos criar condições mais complexas e expressivas em nossos programas. Eles serão fundamentais para a construção de sistemas que tomam decisões baseadas em múltiplos fatores.
