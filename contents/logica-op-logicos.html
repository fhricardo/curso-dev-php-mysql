Aula 09 - Operadores Lógicos

Operador Lógico NOT (Não)

Usados para combinar condições em expressões booleanas.

  • && → E (verdadeiro se todas as condições forem verdadeiras).
  • || → Ou (verdadeiro se pelo menos uma condição for verdadeira).
  • ! → Negação (inverte o valor lógico).

Exemplo:

<?php
    $idade = 20;
    $temCarteira = true;

    if ($idade >= 18 && $temCarteira) {
        echo "Pode dirigir!";
    } else {
        echo "Não pode dirigir.";
    }
?>

Os operadores são a base para expressar cálculos e condições em nossos programas. A partir da próxima aula, vamos utilizar esses operadores em estruturas de decisão (if, else, switch) e construir programas interativos.
